These instructions are for Debian 12. Additional Windows instructions are
below. These instructions are similar for any of our games using Shim, only
you will not use twinkle and will usually use wedge too.

1) sudo apt install libsdl2-dev libfreetype-dev libglm-dev cmake git libflac-dev libogg-dev libvorbis-dev libpng-dev

*1.1*) Most distros don't ship with SDL3 yet, so compile and install SDL3
and SDL3_ttf from source

2) git clone https://codeberg.org/nooskewl/tgui6.git
   git clone https://codeberg.org/nooskewl/shim5.git
   git clone https://codeberg.org/nooskewl/booboo.git
   git clone https://codeberg.org/nooskewl/twinkle.git

3) cd twinkle
   g++ -c -O2 twinkle.cpp
   ar rc libtwinkle.a twinkle.o
   ranlib libtwinkle.a

--

This shell script is used to run cmake... change the paths (trent/code)
to suit your paths.

build.sh:

cmake .. -DCMAKE_BUILD_TYPE=Release -DUSER_INCLUDE_PATH="/home/trent/code/tgui6/include;/home/trent/code/shim5/include;/usr/local/include/SDL3;/usr/local/include/SDL3_ttf;/usr/include/freetype2;/home/trent/code/twinkle" -DUSER_LIBRARY_PATH="/home/trent/code/tgui6/build;/home/trent/code/shim5/build;/home/trent/code/twinkle"

--

4) cd tgui6
   mkdir build
   cd build
   sh ../../build.sh
   make -j

5) cd shim5
   mkdir build
   cd build
   sh ../../build.sh
   make -j

6) cd booboo
   mkdir build
   cd build
   sh ../../build.sh
   make -j

7) You now have BooBoo.exe and BooBooCLI.exe (or without exe on Linux.)
   Remember that BooBoo.exe needs an app, the default is booboo.git/data
   which is a default launcher.

--

Windows specifics:

build.sh:

cmake .. -DCMAKE_TOOLCHAIN_FILE=/home/trent/code/Toolchain-mingw64.cmake -DCMAKE_BUILD_TYPE=Release -DUSER_INCLUDE_PATH="/home/trent/code/twinkle:/home/trent/code/win-include64:/home/trent/code/tgui6/include:/home/trent/code/shim5/include:/home/trent/code/shim5/external:/home/trent/code/win-include64/SDL3" -DUSER_LIBRARY_PATH="/home/trent/code/twinkle:/home/trent/code/win-lib64:/home/trent/code/tgui6/build:/home/trent/code/shim5/build" -DGAME_LAUNCHER=on -G "Unix Makefiles"

You can download all the dependencies (this has everything for all our games too) from:

https://nooskewl.com/files/win-deps64-20250624.zip

This also has a file called Toolchain-mingw64.cmake that you will need to have present.

You need 64 bit MSYS2 shell, 64 bit gcc/cmake and msys/make from pacman.
